Introduction

An algorithm is a finite sequence of steps used to solve a problem.

Algorithms are fundamental to computer science.

  • Step-by-step process
  • Finite instructions
  • Clear logic
  • Input handling
  • Output generation

START
Read number
If number mod 2 = 0
  Print "Even"
Else
  Print "Odd"
END

Characteristics of an algorithm

A good algorithm must be clear and finite.

Read number

  Print "Positive"

END
Types of algorithms

Searching algorithms help find data.

Recursive algorithms solve problems using self-calls.


FUNCTION factorial(n)
If n = 1
  Return 1
Else
  Return n * factorial(n - 1)
END FUNCTION
